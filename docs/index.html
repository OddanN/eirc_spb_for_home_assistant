ЕИРЦ Санкт-Петербурга в Home Assistant

Интеграция для чтения данных по лицевым счетам прямо в вашей панели.

Проект подключает Home Assistant к личному кабинету ЕИРЦ Санкт-Петербурга и поднимает сенсоры по данным квартиры, начислениям и связанным метрикам.

Что умеет

Проект уже умеет авторизоваться и формировать сенсоры по данным помещений.

Авторизация в ЕИРЦ

В проекте заложена работа с авторизацией пользователя и получением текущего профиля через API личного кабинета.

Получение лицевых счетов

Интеграция запрашивает список счетов и детали по каждому счёту, чтобы дальше разложить эти данные на сущности Home Assistant.

Сенсоры по данным помещения

Сейчас формируются сенсоры из блока «Информация о жилом помещении», включая значения и служебные атрибуты.

Установка

Подключение без лишней ручной настройки.

Через HACS

  1. Добавьте репозиторий как кастомный с типом Integration.
  2. Установите EIRC SPB.
  3. Перезапустите Home Assistant.

Вручную

  1. Скопируйте папку custom_components/eirc_spb_for_home_assistant.
  2. Поместите её в конфигурацию Home Assistant.
  3. Перезапустите Home Assistant и добавьте интеграцию через интерфейс.
